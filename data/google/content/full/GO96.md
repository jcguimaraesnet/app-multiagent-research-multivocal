# Tutorials [¶](https://gepa-ai.github.io/gepa/tutorials/\#tutorials "Permanent link")

Welcome to the GEPA tutorials! These hands-on notebooks will help you learn GEPA through practical examples.

## Available Tutorials [¶](https://gepa-ai.github.io/gepa/tutorials/\#available-tutorials "Permanent link")

### DSPy Full Program Evolution [¶](https://gepa-ai.github.io/gepa/tutorials/\#dspy-full-program-evolution "Permanent link")

Learn how to use GEPA to evolve entire DSPy programs, including custom signatures, modules, and control flow logic.

- **[DSPy Full Program Evolution](https://gepa-ai.github.io/gepa/tutorials/dspy_full_program_evolution/)** \- Evolve a complete DSPy program from a basic `ChainOfThought` to a sophisticated multi-step reasoning system. This tutorial demonstrates how GEPA can improve a program from 67% to 93% accuracy on the MATH benchmark.

### ARC AGI Example [¶](https://gepa-ai.github.io/gepa/tutorials/\#arc-agi-example "Permanent link")

- **[ARC AGI Example](https://gepa-ai.github.io/gepa/tutorials/arc_agi/)** \- Apply GEPA to the ARC (Abstraction and Reasoning Corpus) challenge, demonstrating how to optimize programs for complex reasoning tasks.

### 3D Unicorn Optimization (Seedless) [¶](https://gepa-ai.github.io/gepa/tutorials/\#3d-unicorn-optimization-seedless "Permanent link")

Use GEPA's `seed_candidate=None` mode to evolve a complete Python program (build123d + pyrender) that generates a 3D unicorn — starting from no code at all. This tutorial demonstrates how to optimize when you know _what good looks like_ (evaluation aspects) but don't know _how to get started_ (no seed). GEPA bootstraps the first candidate from a natural-language objective, then iteratively refines it using VLM feedback on rendered multi-view images.

- **[3D Unicorn Optimization](https://gepa-ai.github.io/gepa/tutorials/3d_unicorn_optimization/)** \- From no code to a 600+ line 3D modeling program through seedless optimization with visual feedback.

### ConfidenceAdapter AG News Tutorial [¶](https://gepa-ai.github.io/gepa/tutorials/\#confidenceadapter-ag-news-tutorial "Permanent link")

A hands-on notebook for classification-only GEPA optimization using AG News. You can run the full tutorial end-to-end with `DefaultAdapter` versus `ConfidenceAdapter`, inspect GEPA prompt evolution, and reproduce confidence-based metrics and charts.

- **[ConfidenceAdapter Classification Tutorial](https://gepa-ai.github.io/gepa/tutorials/confidence_adapter_classification/)** \- Compare default and confidence-aware classification optimization, from dataset split to AG News per-class and calibration plots.

### LangChain Adapter [¶](https://gepa-ai.github.io/gepa/tutorials/\#langchain-adapter "Permanent link")

## Running Tutorials Locally [¶](https://gepa-ai.github.io/gepa/tutorials/\#running-tutorials-locally "Permanent link")

To run these tutorials locally:

```
# Install GEPA with full dependencies
pip install gepa[full]

# Install Jupyter
pip install jupyter

# Start Jupyter
jupyter notebook
```

Then navigate to the tutorial notebook you want to run.

## Prerequisites [¶](https://gepa-ai.github.io/gepa/tutorials/\#prerequisites "Permanent link")

Before starting the tutorials, ensure you have:

1. **API Keys**: Most tutorials require an OpenAI API key (or other LLM provider)




```
export OPENAI_API_KEY="your-key-here"
```

2. **Python Environment**: Python 3.10+ with GEPA installed




```
pip install gepa[full]
```

3. **Optional**: Install DSPy for the DSPy-specific tutorials




```
pip install dspy
```
