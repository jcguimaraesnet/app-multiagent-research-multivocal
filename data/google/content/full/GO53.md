📅 Session: Scene synthesis

📖 Title: GenAssist: Interactive Prompt-Driven XR Program Generation

👨‍🏫 Authors: Sruti S Srinidhi: Carnegie Mellon University; Akul Singh: Carnegie Mellon University; Edward Lu: Carnegie Mellon University; Anthony Rowe: Carnegie Mellon University

📜 Abstract:  
GenAssist is a system for generating interactive Extended Reality (XR) programs from natural language. At its core is a closed-loop feedback mechanism that uses virtual cameras to iteratively evaluate and refine generated programs, mimicking how human developers compile and interactively test code. To support accurate generation, GenAssist integrates Retrieval-Augmented Generation, retrieving relevant documentation and example code to guide large language models in producing executable, hot-pluggable XR programs in real time. The system supports object placement, animation, and user interaction. GenAssist produces high-quality XR programs, achieving strong accuracy across diverse prompts while remaining fast and usable in practice.
