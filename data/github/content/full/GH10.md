## Model Self-Evolution

[Permalink: Model Self-Evolution](https://github.com/MiniMax-AI/MiniMax-M2.7#model-self-evolution)

M2.7 initiates a cycle of model self-evolution: during development, we let the model update its own memory, build dozens of complex skills for RL experiments, and improve its own learning process based on experiment results. An internal version of M2.7 autonomously optimized a programming scaffold over 100+ rounds — analyzing failure trajectories, modifying code, running evaluations, and deciding to keep or revert — achieving a **30% performance improvement**. On MLE Bench Lite (22 ML competitions), M2.7 achieved a **66.6% medal rate**, second only to Opus-4.6 and GPT-5.4.

[![](https://github.com/MiniMax-AI/MiniMax-M2.7/raw/main/figures/agent_harness.png)](https://github.com/MiniMax-AI/MiniMax-M2.7/blob/main/figures/agent_harness.png)

## Professional Software Engineering

[Permalink: Professional Software Engineering](https://github.com/MiniMax-AI/MiniMax-M2.7#professional-software-engineering)

M2.7 delivers outstanding real-world programming capabilities spanning log analysis, bug troubleshooting, refactoring, code security, and machine learning. Beyond code generation, M2.7 demonstrates strong system-level reasoning — correlating monitoring metrics, conducting trace analysis, verifying root causes in databases, and making SRE-level decisions. Using M2.7, we have reduced live production incident recovery time to **under three minutes** on multiple occasions.

On SWE-Pro, M2.7 achieved **56.22%**, matching GPT-5.3-Codex, with even stronger performance on real-world engineering benchmarks: **SWE Multilingual (76.5)** and **Multi SWE Bench (52.7)**. On **VIBE-Pro (55.6%)**, M2.7 is nearly on par with Opus 4.6. On **Terminal Bench 2 (57.0%)** and **NL2Repo (39.8%)**, M2.7 demonstrates deep understanding of complex engineering systems. M2.7 also supports native **Agent Teams** for multi-agent collaboration with stable role identity and autonomous decision-making.

[![](https://github.com/MiniMax-AI/MiniMax-M2.7/raw/main/figures/agent_teams.gif)](https://github.com/MiniMax-AI/MiniMax-M2.7/blob/main/figures/agent_teams.gif)[![agent_teams.gif](https://github.com/MiniMax-AI/MiniMax-M2.7/raw/main/figures/agent_teams.gif)](https://github.com/MiniMax-AI/MiniMax-M2.7/blob/main/figures/agent_teams.gif)[Open agent_teams.gif in new window](https://github.com/MiniMax-AI/MiniMax-M2.7/blob/main/figures/agent_teams.gif)

## Professional Work

[Permalink: Professional Work](https://github.com/MiniMax-AI/MiniMax-M2.7#professional-work)

## Local Deployment Guide

[Permalink: Local Deployment Guide](https://github.com/MiniMax-AI/MiniMax-M2.7#local-deployment-guide)

Download the model from HuggingFace repository: [https://huggingface.co/MiniMaxAI/MiniMax-M2.7](https://huggingface.co/MiniMaxAI/MiniMax-M2.7)

We recommend using the following inference frameworks (listed alphabetically) to serve the model:

### SGLang

[Permalink: SGLang](https://github.com/MiniMax-AI/MiniMax-M2.7#sglang)

We recommend using [SGLang](https://docs.sglang.io/) to serve MiniMax-M2.7. Please refer to our [SGLang Deployment Guide](https://github.com/MiniMax-AI/MiniMax-M2.7/blob/main/docs/sglang_deploy_guide.md).

### vLLM

[Permalink: vLLM](https://github.com/MiniMax-AI/MiniMax-M2.7#vllm)

We recommend using [vLLM](https://github.com/vllm-project/vllm) to serve MiniMax-M2.7. Please refer to our [vLLM Deployment Guide](https://github.com/MiniMax-AI/MiniMax-M2.7/blob/main/docs/vllm_deploy_guide.md).

### Transformers

[Permalink: Transformers](https://github.com/MiniMax-AI/MiniMax-M2.7#transformers)

We recommend using [Transformers](https://github.com/huggingface/transformers) to serve MiniMax-M2.7. Please refer to our [Transformers Deployment Guide](https://github.com/MiniMax-AI/MiniMax-M2.7/blob/main/docs/transformers_deploy_guide.md).

### ModelScope

[Permalink: ModelScope](https://github.com/MiniMax-AI/MiniMax-M2.7#modelscope)

You also can get model weights from [modelscope](https://modelscope.cn/models/MiniMax/MiniMax-M2.7).

### NVIDIA NIM

[Permalink: NVIDIA NIM](https://github.com/MiniMax-AI/MiniMax-M2.7#nvidia-nim)

MiniMax M2.7 is also available on [NVIDIA NIM Endpoint](https://build.nvidia.com/).

### Inference Parameters

[Permalink: Inference Parameters](https://github.com/MiniMax-AI/MiniMax-M2.7#inference-parameters)

We recommend using the following parameters for best performance: `temperature=1.0`, `top_p = 0.95`, `top_k = 40`. Default system prompt:

```
You are a helpful assistant. Your name is MiniMax-M2.7 and is built by MiniMax.
```
